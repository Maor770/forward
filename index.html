Underwriting Defaults - Edit Mode

Hidden control panel for managing the calculator's default values. Three options below: save current values to your browser as your local default, export current values as JSON to send to Claude for permanent code update, or restore the original code defaults.

Save Current Values as Local Default

Stores the current 22 input values in your browser. Next time you open this file (in this browser), these values will load instead of the code defaults. Cleared if you clear browser data.

Export JSON for Permanent Update

Exports the current values as a JSON snippet. Copy and send to Claude - Claude will embed it as the new code default for all future versions of this file. This is the way to make the values "stick" forever.
